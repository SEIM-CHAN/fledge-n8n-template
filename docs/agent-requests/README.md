# Agent Requests

ここは、人間がエージェントに対して「何を作ってほしいか」を置く場所です。

最初の依頼は [完成例](example-instagram-dm-auto-reply.md) を複製して書き換えるのが早いです。

## 使い方

1. 新しい依頼ごとに Markdown ファイルを 1 つ作る
2. 目的、入力、出力、制約、完成条件を書く
3. そのファイルをエージェントに読ませて、`workflows/n8n/` へ JSON を作らせる

## 書いておくと良い項目

- 背景
- 解きたい業務課題
- トリガー
- 入力データ
- 必要な外部サービス
- 保存したいデータ
- 期待する出力
- NG事項
- 完成条件

## Template

`request-template.md` をコピーして使う想定です。
